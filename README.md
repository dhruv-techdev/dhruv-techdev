### 🛒 [ShopSphere — Microservices E-Commerce Backend](https://github.com/dhruv-techdev/ShopSphere-Microservices-E-Commerce)

An **8-service production-grade backend** (Gateway, User, Product, Cart, Order, Inventory, Payment, Notification) with Eureka service discovery, a centralized API Gateway, and a **Kafka async event pipeline** for order workflows. Schema-per-service DB isolation, JWT RBAC, Redis-backed cart, and per-service OpenAPI 3 specs — delivered across **32 Jira user stories** with full GitFlow and CI gates.

### ☁️ [Fragments — Cloud-Native Microservice](https://github.com/dhruv-techdev/fragments-public)

A REST microservice that stores, retrieves, and **converts text/image fragments** (Markdown→HTML, PNG→WebP). Dual-environment design — in-memory for dev, **S3 + DynamoDB** for prod — switchable via env vars with **zero code changes**. Full **CI/CD to AWS ECS** (GitHub Actions → ECR → ECS), owner-isolated data access, 16 unit test files + 14 Hurl integration tests, and graceful shutdown handling.
